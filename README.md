如何使用：

1、修改 run.exp  bc_path 这个变量，指的是bitshares_client的基本路径，及你的钱包名。

2、
#screen -S bcs
#cd /data/src/bitsharesx/programs/client/
#./run.exp

3、转入相应钱包的密码

4、ctrl+a+d 让screen进入后台运行。

5、可以使用screen -r bcs 进入之前的screen
